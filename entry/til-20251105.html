今日の学び: TIL というものを知った

X でこんなポストを見かけた。

GitHub を活用して、
日々の学習の記録をつけていく習慣が、TIL

TIL（Today I Learned：今日学んだこと）は、
世界中の開発者の間で、10年以上前から行われている取り組み

学習の記録を GitHub リポジトリで管理することで、

- 単純に、記録を書くことで理解が深まる
- Git/GitHub… pic.twitter.com/pz9gxnGEG0

— kazzyfrog (@kazzyfrog) November 4, 2025

なるほど、アウトプットするまでのハードルを減らすことで習慣化に結びつけようという意図らしい。投稿した人のブログ記事を読むと 10 年以上前からある文化のようだ。

常にアウトプット不足なので自分も初めてみようと思った。

GitHub に TIL 用の repository を作るのが王道のやり方らしいが、せっかくこのブログがあるのでとりあえずこれを使う。記事のボイラープレートを展開するスクリプトを作って PR まで立てられるようにしたらスマホの GitHub アプリのみで投稿までできそうだ。夢が広がるね。
