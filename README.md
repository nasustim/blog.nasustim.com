# blog.nasustim.com

https://blog.nasustim.com

## Setup

Developing this project requires [mise](https://mise.jdx.dev/)

Installing runtimes it requires

```bash
$ mise install
```

Installing npm packages

```bash
$ bun install
```

## Usage

Start dev server

```bash
$ bun run
```

Format code style

```bash
$ bun run biome:fix
$ bun run textlint:fix
```
